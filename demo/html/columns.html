Basic Column Definition

If you do not wish to display all the properties of the objects bound to the grid, you can define the databinding for each column. You also have the option of turning off sorting and filtering per column, plus setting up the width of the column, in any kind of measurement unit you wish.
Generic Columns

Extra columns can be added in the body, even without defining the header. Tapping into the currently bound row item can be done through the gridItem scope property. More examples that are covering this topic are available in the templates section.
