Customizations

Customizations can occur at the header, body and footer level.

One very important rule to remember for the templates of the data cells (the ones inside the TBODY element) is to always provide the field-name attribute if they refer to a column which is data bound by the grid. trNgGrid will match these cells with the ones in the header, either automatically produced (if the THEAD element was empty or did not exist) or manually defined and marked with the same field-name attribute.
The cells part of custom columns don't require any special attention. They will be matched with the position of the corresponding cells in the header.
