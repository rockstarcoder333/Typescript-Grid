Paging

By default, all the items in the data array are displayed on a single page in the grid. Scrollbars can be manually added through CSS.
If you wish to activate paging, you must set the page-items attribute. Paging is applied after the data was filtered through the global search filter, the column search filters, and finally after applying sorting.
Selections

Selections are enabled by default. They are controlled by the enable-selections and the enable-multi-row-selections attributes.

If you wish to watch or manipulate the grid selections, you can assign a scope variable to the selected-items attribute. That will trigger the two-way data binding.
